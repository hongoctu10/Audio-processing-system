# Audio Processing System

## Description
This project focuses on speech signal processing, including pitch shifting and time scaling.

## Features
- Adjust speech speed (time scaling)
- Modify pitch of audio signals
- Display waveform and spectrogram
- Simple graphical user interface

## Technologies
- MATLAB
- Signal Processing techniques

## How to run
1. Open MATLAB
2. Run file `main.m`
3. Use the interface to process audio
